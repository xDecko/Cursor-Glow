# Cursor-Glow

Cursor-Glow  allows you to create an impressive light effect that follows the cursor on web pages. The effect generates a circle with a blur effect, creating the illusion of a light illuminating the screen as the cursor moves.


Features:

Easy to implement on any web page.
Customization of the size, color, and opacity of the light circle.
Configuration of the blur to adjust the intensity of the illumination.
Lightweight and fast, without significantly affecting the page's performance.
